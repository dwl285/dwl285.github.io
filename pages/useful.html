Git workflow

Git: The Simple Guide - Does a good job of summarising the most important aspects

Here is my even more simple description of the steps required:

  • step 1: git clone https://github.com/dwl285/portfolio-site
  • step 2: make changes
  • step 3: git status (to see what files have changed)
  • step 4: git add filename1 filename 2
  • step 5: git status (check you've added all the files)
  • step 6: git commit -m "describe the change"
  • step 7: git push origin master
  • step 8: check github to see if your files are there
